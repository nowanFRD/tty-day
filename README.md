#tty-day by nowanFRD

/Hi this is a simple bash script for terminal kinda like tty-clock or stuff, it use figlet to make ASCII-art
/To do app will work you need figlet (debian - sudo apt install figlet, arch - sudo pacman -S figlet, fedora - sudo dnf install figlet...)
			

			HOW TO INSTALL THE MAIN SCRIPT
'''sudo cp tty-day /usr/local/bin/tty-day
   sudo chmod +x /usr/local/bin/tty-day'''
			HOW TO START
-tty-day
